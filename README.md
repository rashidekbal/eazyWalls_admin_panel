# EazyWalls Admin Client

## 1. Introduction

A React-based administration panel for managing wallpapers, categories, and featured/trending collections.

## 2. Technology Stack

- **Framework**: React.js (v19)
- **Build Tool**: Vite
- **Styling**: CSS Modules
- **HTTP Client**: Axios

## 3. Setup & Installation

1. Navigate to the `adminClient` directory:
   ```bash
   cd adminClient
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
   Access at `http://localhost:5173`.

## 4. Key Components

- **`Store.jsx`**: Manages global state and API interactions.
- **`src/assets/page/`**: Application views (Dashboard, AddWallpaper, etc.).
- **`src/assets/components/`**: Reusable UI components.

For API details, refer to the **Server Documentation**.
